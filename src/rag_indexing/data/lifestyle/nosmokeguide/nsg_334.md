# 금연과 음악치료_조작된 리듬연주(펜비트)

펜비트 즉흥리듬 연주해보기
간단히 펜을 이용하여 리듬즉흥연주 할 수 있습니다. 손에 든 펜을 이용하여 간단한 리듬부터 배워 일상에서 활용해 봅시다.
펜 비트 기본 기술A(스냅) 펜을 잡은 상태로 손목을 바닥에 쳐주세요 동영상 보기
B(탭)잡은 펜 엄지 쪽의 끝으로 바닥을 쳐주세요. 동영상 보기
C(플릭) 펜의 전체로 바닥을 치는 쳐주세요. 동영상 보기

펜 비트 응용 기술D(노크) 손가락 관절로 노크하듯이 바닥을 쳐주세요. 동영상 보기
E(리틀탭) 탭의 반대로, 펜의 새끼손가락 쪽의 끝으로 바닥을 쳐주세요. 동영상 보기
F(리버스플릭) 리틀탭과 플릭을 합친 기술, 리틀탭 상태에서 플릭을 쳐 소리내주세요. 동영상 보기
G(스틱) 리틀탭에 큰 동작을 섞은 것으로, 리틀탭 상태에서 90°로 세운 다음 바닥을 쳐주세요. 동영상 보기
Sb(탭스크래치) 탭을 친 상태로 바닥을 그어서 소리를 내주세요. 동영상 보기
Sc(플릭스크래치) 플릭을 친 상태로 바닥을 쓸어서 소리를 내주세요. 동영상 보기
O(공박) 소리를 내지 않습니다.
