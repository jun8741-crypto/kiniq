# 문제 정의 — 만성신부전(CKD) 조기 발굴·생활습관 중재

> **본 문서의 모든 통계는 공개 자료(질병관리청, KOSIS, 대한신장학회, KDIGO 가이드라인) 기반이며, 의료 진단·치료를 대체하지 않습니다.**

---

## 1. 문제의 대상 — 누가, 얼마나

### 1-1. 한국 CKD 환자 규모

| 지표 | 수치 | 출처 |
|---|---|---|
| 한국 성인 CKD 유병률 | **약 8.2%** (성인 7명 중 1명) | KNHANES 2021, 대한신장학회 Fact Sheet |
| 한국 CKD 진단자 수 (추정) | **약 380만 명** | 보건복지부 2022 |
| 30세 이상 성인 중 CKD 인지율 | **9.6%** (10명 중 1명만 본인 인지) | 대한신장학회 2023 |
| 말기신부전(ESRD) 신규 환자 | 연간 **약 17,000명** | 대한신장학회 ESRD Registry 2022 |

### 1-2. 핵심 위험 인자 (Model 1 입력 변수)

- **고혈압**: CKD 진행의 1위 원인 (50%+)
- **당뇨병**: 2위 원인 (30%+) — 당뇨병성 신증
- **고령**: 65세 이상에서 유병률 **20%+**
- **비만·복부비만**: BMI ≥ 25 시 위험 ↑

### 1-3. 인지율 격차의 사회적 영향

CKD는 **무증상 진행**(eGFR 60 이하부터 자각 증상 나타나기 시작) 특성상, **G3 단계까지 본인이 모르는 경우가 90%+**. 조기 발굴 실패로 인한:

- 투석 진행 시 **연간 의료비 1인당 약 3,000만 원**
- 신장 이식 대기 평균 **5~7년**
- 사회적 비용(생산성 손실 포함) 연간 **약 3조 원**

---

## 2. 영향 범위 — 왜 지금 해결해야 하나

### 2-1. 한국 인구 구조와 시장

- **40세 이상 인구**: 약 **2,800만 명** (2024 기준, 통계청)
- 그중 CKD 위험군(고혈압·당뇨 보유) 약 **30%+** = **840만 명**
- 디지털 헬스케어 시장 한국 규모: **2024년 약 4조 원**, 연 **15% 성장** (한국보건산업진흥원)

### 2-2. 정책적 배경

- **2022년 보건복지부**: 만성질환 예방·관리 사업 5개년 계획 발표
- **2023년 식약처**: 디지털 치료기기 가이드라인 발간 → 만성질환 행동변화 앱 인허가 가능
- **국민건강보험공단** 일반 건강검진 결과지 데이터 표준화 (OCR 자동화 가능)

### 2-3. 의학적 근거

- **KDIGO 2012 가이드라인**: G3a/G3b 분리 (eGFR 30~59를 둘로 쪼개 합병증 위험 차별화)
- **CKD-EPI 2021 공식**: 인종 보정 제거된 표준 eGFR 계산식
- **생활습관 중재 효과 (Lancet 2020)**: 식단·운동·금연 통합 중재로 **CKD 진행 속도 30% 감소** 입증

## 3. 시장 분석 — 경쟁·차별점

### 3-1. 기존 헬스케어 앱과의 차별

| 앱 카테고리 | 강점 | 약점 (우리가 채우는 부분) |
|---|---|---|
| **삼성헬스·애플건강** | 기본 데이터 수집, 보급률 ↑ | CKD 특화 분석 X, 행동 변화 미약 |
| **눔(Noom)** | 식단 코칭, 행동변화 이론 적용 | 신장 기능 추적 X, 의료 데이터 미반영 |
| **닥터마인드·강남세브란스 앱** | 의료기관 연계 | 일반 사용자 진입 장벽 ↑ |
| **우리 서비스** ⭐ | **CKD 위험 자동 분석 + 생활습관 챌린지 + 의학 표준(KDIGO) 준수** | — |

### 3-2. 우리 서비스의 핵심 가치

1. **조기 발굴**: 건강검진 결과만 입력하면 ML이 G1~G4 그룹 자동 배정 (Model 1)
2. **개인화 추천**: SHAP 기반 위험 기여도 상위 3개 변수 추출 (Model 2)
3. **행동 변화 유도**: 챌린지·게이미피케이션(3단계 진화·5종 캐릭터)으로 21일 습관 형성 임계 돌파
4. **의료 안전성**: KDIGO 표준 준수 + 의료법 금지 표현 차단 + 모든 페이지 면책 푸터
5. **고령 친화**: 큰 글씨·간단 메뉴(3~5단계 접근)·한국어 에러 메시지 일관

## 4. 우리가 풀려는 구체적 문제

### 4-1. 핵심 가설 (Problem Statement)

> "한국 40세 이상 성인의 8.2%가 CKD를 갖고 있지만 **90%가 본인 인지 X**.
> 건강검진 결과를 활용해 위험을 조기 발굴하고, **재미있는 챌린지로 21~100일 행동 변화 임계**를 넘기게 하면, **CKD 진행 속도 30% 감소** 효과를 디지털로 재현할 수 있다."

### 4-2. 측정 가능한 성공 지표 (KPI)

| 영역 | 지표 | 목표 |
|---|---|---|
| **발굴** | 위험군 자동 분류 정확도 (Model 1 AUROC) | ≥ 0.80 (5-fold CV) |
| **행동 변화** | 7일 리텐션 | ≥ 50% (데모 사용자 기준) |
| **사용자 경험** | API P95 응답 시간 | < 3,000ms (실측 **18ms** 달성) |
| **의료 안전** | 의료법 금지 표현 노출 | **0건** (현재 달성) |

### 4-3. 범위 (Scope)

✅ **MVP 범위 (이번 발표)**:
- 회원가입·로그인(JWT)·비밀번호 재설정(이메일 인증)
- 건강검진 수동 입력·G1~G4 그룹 fallback 분기(ML 없을 때)
- 5종 카테고리 챌린지(수분·운동·식단·수면·스트레스) + 체크인
- 3단계 캐릭터 진화 + 5종 종 추첨 + 숙련도 배경 4종
- 8종 대시보드 위젯(계기판·추세·잔디 히트맵·라디알·시뮬레이션 등)
- 의료 안전 가드(응급 119·자해 1393·약물·면책 푸터)

⏭️ **차기 마일스톤 (범위 외)**:
- OCR 자동 입력 (AI 팀)
- Model 1·2 실제 학습 (AI 팀)
- LLM 기반 개인 추천 (AI 팀)
- 다크모드, 부분 PWA, 푸시 알림
