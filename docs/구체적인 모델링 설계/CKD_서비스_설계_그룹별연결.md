# CKD 서비스 전체 설계 — 모델·그룹·기능 연결

---

## 1. Model 1 — 스크리닝 AI

### 무엇을 하는 모델인가
OCR + 설문으로 입력된 전체 피처로 **이 사람이 CKD(eGFR < 60) 위험군인지** 확률 점수를 출력한다.
eGFR/creatinine은 타겟 계산에만 쓰이고 모델 입력에서는 제외.

### 입력 피처
전체 피처 (임상 수치 + 생활습관 + 진단명 + 가족력)  
→ eGFR, creatinine만 제외

### 출력
- CKD 위험 확률 점수 (0~1)
- 이 점수가 **G3 분류 임계값** 기준이 됨 (Youden Index로 학습 후 결정)

### 사용자 화면 예시
```
┌─────────────────────────────────────┐
│  신장 건강 주의 지수                 │
│                                     │
│  또래(50대 여성) 대비 상위 18%       │
│  ████████░░░░░░░░░░  (위험 구간)    │
│                                     │
│  주요 영향 요인 →  [자세히 보기]    │
└─────────────────────────────────────┘
```
> "CKD 위험 XX%" 표현 금지 (의료법)  
> "또래 대비 상위 X%" 표현으로 대체

## 2. Model 2 — 생활습관 AI

### 무엇을 하는 모델인가
진단명과 혈압·혈당 수치를 제외한 **행동 가능한 변수만**으로 CKD 위험을 학습.
목적은 예측 정확도가 아닌 **Local SHAP을 통한 개인 맞춤 행동 조언**.

### 입력 피처 (제외 항목 명시)

| 제외 | 이유 |
|------|------|
| htn_diagnosed, dm_diagnosed, dyslipidemia_diagnosed | 진단명 — 질병 라벨, 행동 아님 |
| sbp, dbp | 결과 지표 — SHAP 지배 + "혈압 낮추세요"는 목표지 행동 아님 |
| fasting_glucose | 동일 이유 |
| eGFR, creatinine | 누수 변수 |

| 포함 (핵심) | 역할 |
|------------|------|
| LDL, triglycerides, HDL | 식이 대리 변수 (식이 데이터 없어서 대체) |
| waist_cm, BMI, waist_height_ratio | 체중·복부비만 행동 조언 |
| smoking_current, drinking_freq | 직접 행동 변수 |
| sitting_hours, walking_days, vigorous/moderate 변수 | 운동·활동 행동 변수 |
| hemoglobin | 철분·영양 상태 (anemia 이진형 제거, 연속형 유지) |
| 가족력 5개 | 맥락 레이어 (조언 아닌 설명용) |
| age, gender | 맥락 레이어 |

### 출력
각 사용자별 **Local SHAP 값** → 어떤 변수가 이 사람의 CKD 위험에 얼마나 기여하는지

### 사용자 화면 예시
```
┌─────────────────────────────────────────┐
│  지금 신장에 영향을 주는 생활습관       │
│                                         │
│  위험을 높이는 것          위험을 낮추는 것 │
│  ▶ 복부둘레    ████  44%   ▶ 걷기     ██  6% │
│  ▶ 흡연        ██   10%   ▶ HDL      ██  5% │
│  ▶ 좌식시간    ██    8%                  │
│                                         │
│  "나이와 가족력(고혈압)을 고려했을 때,  │
│   지금 가장 먼저 바꿀 수 있는 건        │
│   복부둘레 감소입니다."                 │
└─────────────────────────────────────────┘
```

## 3. Risk Score — 2레이어 위험도 표시

### 설계 목적
Model 2의 Local SHAP 값을 분해하여 **바꿀 수 없는 위험(Baseline)**과 **바꿀 수 있는 위험(Modifiable)**을 분리 표시한다.  
사용자가 생활습관 개선의 실질적 효과를 시각적으로 확인할 수 있게 하는 것이 목적이며, 단순 확률 숫자보다 "무엇을 바꾸면 어떻게 달라지는가"를 직관적으로 전달한다.

### 2레이어 구조

| 레이어 | 구성 변수 | 의미 |
|--------|----------|------|
| **Baseline Risk** | age, gender, family_dm, family_htn, family_dyslipidemia, family_ihd, family_stroke | 나이·성별·가족력 — 행동으로 바꿀 수 없는 위험 |
| **Modifiable Risk** | 위 7개 제외 나머지 생활습관 변수 SHAP 합계 | 식이·운동·흡연·체중 등 — 행동으로 줄일 수 있는 위험 |

Total Risk Score = Baseline Risk + Modifiable Risk (Model 2 출력 확률과 동일)

### SHAP 분해 구현

```python
# Model 2 TreeSHAP 결과에서 두 레이어 분리
baseline_vars = [
    'age', 'gender',
    'family_dm', 'family_htn', 'family_dyslipidemia', 'family_ihd', 'family_stroke'
]
modifiable_vars = [v for v in shap_feature_names if v not in baseline_vars]

# base_value: 학습 데이터 전체 평균 로그오즈
baseline_risk = base_value + sum(shap_values[v] for v in baseline_vars)
modifiable_risk = sum(shap_values[v] for v in modifiable_vars)

# sigmoid 변환 → 확률
current_risk = sigmoid(baseline_risk + modifiable_risk)
# 검증: current_risk == model.predict_proba(user_data) 와 동일해야 함
```

### Counterfactual (What-if) 시나리오

사용자의 특정 변수를 가상으로 변경 → LightGBM 재실행 → 확률 비교.  
Baseline Risk는 고정, Modifiable Risk만 재계산하여 차이를 표시한다.

```python
# 예시: 흡연 중단 시뮬레이션
modified_data = user_data.copy()
modified_data['smoking_current'] = 0  # 현재 흡연(1 또는 2) → 비흡연(0)으로 변경

new_shap = explainer.shap_values(modified_data)[0]
new_modifiable_risk = sum(new_shap[v] for v in modifiable_vars)

# Baseline Risk는 변경 없이 그대로 유지
improved_risk = sigmoid(baseline_risk + new_modifiable_risk)
delta = current_risk - improved_risk  # 양수 = 위험 감소
```

> **구현 주의사항**  
> - Counterfactual은 단일 변수 변경 기준으로 표시 (다중 변수 동시 변경 시 상호작용 해석 어려움)  
> - 흡연·음주처럼 0으로 설정 가능한 변수만 What-if 대상으로 제한  
> - 연속형 변수(waist_cm, BMI 등)는 "10% 감소 시" 같은 고정 시나리오로 설정  
> - AutoGluon에서 추출한 LightGBM의 TreeSHAP explainer 재사용 (별도 학습 불필요)

### 표현 가이드라인 (의료법 준수)

| 금지 표현 | 허용 표현 |
|----------|---------|
| "위험이 X% 감소합니다" | "예상 위험 변화: ↓ Xpt" |
| "흡연을 끊으면 CKD가 예방됩니다" | "금연 시 예상 위험 지수 변화" |
| "이 점수가 당신의 CKD 확률입니다" | "생활습관 기반 신장 건강 지수" |
| "현재 위험도 62%" | "신장 건강 주의 지수 62 / 100" |

### 사용자 화면 예시

```

│                                          │
```

## 4. 그룹 분류 기준 (우선순위 G1 → G2 → G3 → G4)

| 그룹 | 분류 조건 | 근거 |
|------|----------|------|
| G1 신장의심군 | OCR creatinine → eGFR < 60 | 단회 측정, 재검 필수 |
| G2 원인질환군 | eGFR ≥ 60 AND (SBP≥130 OR DBP≥80 OR glucose≥100 OR htn_diagnosed=1 OR dm_diagnosed=1) | 규칙 기반 |
| G3 AI고위험군 | G1·G2 미해당 AND Model1 score ≥ 임계값 | Youden Index로 학습 후 결정 |
| G4 일반건강군 | 위 조건 모두 미해당 | 기본값 |

## 5. G1 — 신장의심군 (전체의 약 4%)

### 사용자 프로필
건강검진 creatinine 기반 eGFR < 60. 이미 신장 기능 저하 의심.  
단, 단회 측정이므로 AKI·탈수 등 일시적 원인 가능성도 있음.

### 앱 화면 흐름
```
[메인 상태]
"신장 기능 주의 신호 감지"
단회 측정 결과입니다. 반드시 재검이 필요합니다.

[추가 검사 안내]
신장내과 방문 시 요청하세요:
- 소변 알부민/크레아티닌 비율 (UACR)
- 전해질 (칼륨, 인, 나트륨)
- 혈청 알부민
- 신장 초음파

[LLM 챗봇]
궁금한 것을 물어보세요 (RAG 기반 응답)
```

### Challenge 설계
> 격렬한 운동 챌린지 금지 (신장 부하 증가 위험)

| 챌린지 | 내용 | 기간 |
|--------|------|------|
| 저단백 식단 실천 | 하루 단백질 0.8g/kg 이하 기록 | 7일 |
| 수분 관리 | 하루 물 1.5~2L 유지 (의사 지시 없으면 과다 금지) | 매일 |
| 금연 (흡연자만) | 금연 D-day 설정 | 지속 |
| 병원 예약 | 신장내과 예약 완료 인증 | 1회 |

### RAG 지식베이스 (G1 전용)
- 대한신장학회 만성콩팥병 환자 식이 가이드
- KDIGO 2024 CKD 관리 지침 (한국어)
- 당뇨+CKD 동반 식이 가이드 (dm_diagnosed=1인 경우 우선 검색)
- 만성콩팥병 Q&A (증상, 진행, 일상 주의사항)
- 저단백·저칼륨·저인 식품 목록

### LLM 프롬프트 구조
```
[맥락 레이어]
나이: 65세 / 성별: 여성 / 가족력: 고혈압 있음

[상태]
eGFR: 48 (단회 측정) → G1 분류
당뇨 진단: 있음 → 당뇨+CKD 동반 케이스

[RAG 검색 결과]
당뇨+CKD 동반 식이 가이드 발췌...

[식이설문 4문항 응답]
육류 섭취 빈도: 주 5회 이상

[조건]
- "CKD 확진"이 아닌 "의심" 표현 사용
- 의약품·처방 조언 금지
- 추가 검사 및 병원 방문 권고 포함
- 출처 명시 (대한신장학회 등)
```

### LLM 출력 예시
> "나이와 고혈압 가족력을 고려했을 때, 현재 신장 기능 주의가 필요한 상태입니다.  
> 당뇨와 신장 질환이 동반될 경우 식이 관리가 까다롭습니다. 일반적인 당뇨 식이와 달리, 신장 부담을 줄이기 위해 단백질은 하루 체중(kg) × 0.8g 이하로 제한하고, 칼륨이 많은 바나나·토마토·감자는 줄이는 것이 좋습니다.  
> ※ 출처: 대한신장학회 CKD 식이 가이드  
> 반드시 신장내과에서 UACR 검사를 통해 정확한 상태를 확인하세요."

## 6. G2 — 원인질환군 (전체의 약 30~40% 추정)

### 사용자 프로필
eGFR 정상이지만 고혈압·당뇨·경계 혈압·경계 혈당 보유.  
CKD로 이어질 수 있는 원인 질환 관리가 핵심.

### G2 내부 분기 (강도 조절)
| 세부 분류 | 조건 | 챌린지 강도 |
|----------|------|------------|
| 진단군 | htn_diagnosed=1 OR dm_diagnosed=1 OR SBP≥140 OR glucose≥126 | 고강도 |
| 경계군 | SBP 130~139 OR DBP 80~89 OR glucose 100~125 (진단 없음) | 중강도 |

### 앱 화면 흐름
```
[메인 상태]
"신장을 위협하는 원인 관리 중"

[Model 2 SHAP — 보조 개인화]
"지금 가장 영향 큰 생활습관: 복부둘레 > 좌식시간 > 흡연"
(G2는 그룹 조언이 메인, SHAP은 우선순위 보조로만 활용)

[오늘의 챌린지]
혈압 관리 → 저염 식단 실천
```

### Challenge 설계

**공통**
| 챌린지 | 내용 |
|--------|------|
| 저염 식단 | 하루 나트륨 2,000mg 이하 기록 |
| 유산소 운동 | 주 5회 30분 걷기 |
| 체중 관리 | 주 1회 체중 기록 |

**고혈압 진단군 추가**
| 챌린지 | 내용 |
|--------|------|
| 혈압 자가 측정 | 아침·저녁 2회 기록 |
| 금연 (흡연자) | 금연 일수 카운트 |

**당뇨 진단군 추가**
| 챌린지 | 내용 |
|--------|------|
| 식후 혈당 관리 | 정제탄수화물 줄이기 미션 |
| 식후 10분 걷기 | 혈당 스파이크 예방 |

### RAG 지식베이스 (G2 전용)
- 대한고혈압학회 생활습관 가이드
- ADA 당뇨 식이 및 운동 권고 (2024)
- 저염 식단 레시피 DB
- 혈압·혈당이 신장에 미치는 영향 (일반인 설명용)

### LLM 출력 예시 (58세 남성, 고혈압 진단, 당뇨 경계)
> "나이와 고혈압 이력을 고려했을 때, 현재 혈압 관리가 신장 보호의 핵심입니다.  
> 생활습관 분석 결과, 복부둘레(88cm)와 좌식시간(9시간)이 혈압을 높이는 주요 요인입니다.  
> 우선 나트륨을 하루 2,000mg 이하로 줄이고, 식후 10분 걷기부터 시작해보세요."

## 7. G3 — AI 고위험군 (Model 2 SHAP의 핵심 타겟)

### 사용자 프로필
eGFR 정상, 진단명 없음, 혈압·혈당 수치도 정상 범위.  
그런데 Model 1이 위험 신호를 감지한 그룹.  
**"나는 건강한 줄 알았는데?"** 가 이 그룹의 핵심 페르소나.

### 앱 화면 흐름
```
[메인 상태 — 임팩트 있는 메시지]
"겉으로는 정상, AI는 주의 신호 감지"
현재 특별한 진단은 없지만, 생활습관 분석 결과
신장 건강에 주의가 필요한 상태입니다.

[Model 2 SHAP — 핵심 기능]
"지금 신장에 영향을 주는 것들"
1위. 복부둘레  ████████  40%  → 챌린지 연결
2위. 좌식시간  █████     25%  → 챌린지 연결
3위. 흡연      ████      18%  → 챌린지 연결

[정기 검진 리마인더]
"마지막 건강검진: 11개월 전 — 재검진을 권장합니다"
```

### Challenge 설계 (SHAP 순위 기반 자동 배정)
SHAP 1위 변수 → 첫 번째 챌린지 자동 배정

| SHAP 상위 변수 | 배정되는 챌린지 |
|--------------|--------------|
| waist_cm 높음 | "이번 주 매일 30분 걷기" |
| sitting_hours 높음 | "매 1시간마다 5분 스트레칭" |
| smoking_current = 2 | "금연 D-day 설정 + 금연 Day 1" |
| triglycerides 높음 | "정제탄수화물·당류 줄이기 7일" |
| LDL 높음 | "포화지방 식품 제한 (붉은 고기·버터) 7일" |
| HDL 낮음 | "유산소 운동 주 4회" |
| walking_days 낮음 | "주 5일 걷기 달성" |

### RAG 지식베이스 (G3 전용)
- 생활습관과 CKD 연관성 (일반인 눈높이 자료)
- 복부비만 개선 운동 가이드
- 금연 지원 자료 (금연길라잡이 등)
- 콜레스테롤·중성지방 식이 개선 가이드
- "왜 나는 정상인데 위험하다고 하나요?" FAQ

### LLM 출력 예시 (52세 여성, 진단 없음, SHAP: waist 40% / sitting 25%)
> "52세 여성 기준 또래 대비 신장 건강 주의 지수 상위 15%입니다.  
> 현재 별도 진단은 없지만, 복부둘레와 긴 좌식시간이 신장 건강에 조용히 영향을 주고 있어요.  
> 복부비만은 인슐린저항성을 높여 신장 여과 기능에 부담을 줄 수 있습니다.  
> 가장 먼저: 하루 30분 걷기를 주 5일로 늘려보세요. 오늘 챌린지를 시작할 수 있어요."

## 11. 챌린지 시스템 전체 구조

### 트랙 구분

| 트랙 | 대상 | 특징 |
|------|------|------|
| **A트랙 (케어트랙)** | G1 신장 집중 관리군, G2 신장 위험 관리군 | 임상 기반 제한 적용, 고강도 |
| **B트랙 (일반트랙)** | G3 신장 사전 관리군, G4 건강 습관 형성군 | 예방·습관 형성 중심, 동기 부여 |

### 스테이지 구조 (트랙 공통)

| 단계 | 아이콘 | 특징 |
|------|--------|------|
| 1단계 | 🌱 입문 | 낮은 진입장벽, 작은 행동 변화 |
| 2단계 | 🚶 초보 | 규칙성 형성 |
| 3단계 | 🏃 중급 | 복합 행동, 지속성 요구 |
| 4단계 | 💪 숙련 | 자기관리 루틴화 |

스테이지 상승 시 캐릭터 배경 변경 → 달성감 제공

---

### 카테고리별 스테이지 설계

#### 💧 수분

| 스테이지 | A트랙 (G1·G2) | B트랙 (G3·G4) |
|---------|--------------|--------------|
| 🌱 입문 | 하루 수분 섭취량 기록 시작 | 하루 물 1.5L 마시기 |
| 🚶 초보 | 의사 권고량 범위 내 수분 유지 | 하루 물 2L 달성 |
| 🏃 중급 | 부종·소변량 변화 관찰 일지 | 카페인·음료 대신 물로 대체 |
| 💪 숙련 | 수분·전해질 균형 자기관리 | 수분 섭취 패턴 최적화 |

> ⚠️ G1 주의: CKD 단계에 따라 수분 제한이 필요할 수 있음. "많이 마시기" 지침이 역효과 날 수 있어 A트랙은 "의사 권고량 범위 내"로 표현 필수.

---

#### 🥗 식단

| 스테이지 | A트랙 (G1·G2) | B트랙 (G3·G4) |
|---------|--------------|--------------|
| 🌱 입문 | 저염 식품 1가지 선택 / G1: 저단백 식품 인식 | 나트륨 높은 식품 1가지 줄이기 |
| 🚶 초보 | 나트륨 2,000mg 이하 3일 실천 / G1: 칼륨 높은 식품 제한 | 정제탄수화물 줄이기 7일 |
| 🏃 중급 | 저염+저단백 식단 주 5일 / G2: 혈당 스파이크 식품 제거 | LDL 높으면 포화지방 제한 / TG 높으면 당류 제한 |
| 💪 숙련 | G1: CKD 맞춤 주간 식단 계획 / G2: 혈압·혈당 동시 관리 식단 | 균형 식단 루틴화 |

> G1 당뇨 동반 시: 식단 챌린지에 "당뇨+CKD 동반 가이드" 연결 (저단백 + 혈당 관리 병행)

#### 🏃 운동

| 스테이지 | A트랙 (G1·G2) | B트랙 (G3·G4) |
|---------|--------------|--------------|
| 🌱 입문 | 하루 10분 가벼운 걷기 | 하루 10분 걷기 |
| 🚶 초보 | 주 3회 20분 걷기 | 주 3회 30분 걷기 |
| 🏃 중급 | 주 5회 30분 유산소 | 주 5회 30분 + 계단 이용 |
| 💪 숙련 | 유산소 + 가벼운 근력 병행 | 유산소 + 근력 복합 루틴 |

> ⚠️ G1 주의: 격렬한 운동 제한. 신장 혈류 부담 증가 위험. 입문~초보 단계에서 강도 상한 설정 필요.

#### 😴 수면

| 스테이지 | A트랙 (G1·G2) | B트랙 (G3·G4) |
|---------|--------------|--------------|
| 🌱 입문 | 수면 시간 기록 시작 | 수면 시간 기록 시작 |
| 🚶 초보 | 7시간 수면 목표 3일 달성 | 7시간 수면 목표 5일 달성 |
| 🏃 중급 | 규칙적 취침 시간 유지 (±30분) | 취침 1시간 전 전자기기 제한 |
| 💪 숙련 | 수면 질 개선 루틴 (혈압 관리 연계) | 수면 루틴 완성 |

> 수면 부족은 혈압 상승과 연관. G2는 수면 챌린지를 혈압 관리 맥락으로 연결 가능.

| 스테이지 | A트랙 (G1·G2) | B트랙 (G3·G4) |
|---------|--------------|--------------|

| 🚶 초보 | 하루 5분 복식호흡 | 하루 10분 가벼운 명상 |

| 💪 숙련 | 스트레스-혈압 연관 자기 모니터링 | 스트레스 자기관리 루틴 완성 |

> 만성 스트레스는 코르티솔 상승 → 혈압 증가 → 신장 부담. G2에 특히 연관성 높음.

### SHAP → 챌린지 카테고리 자동 매핑 (G3 핵심)

G3는 SHAP 상위 변수에 따라 첫 번째 챌린지 카테고리가 자동 배정됨.

| SHAP 상위 변수 | 배정 카테고리 | 시작 챌린지 예시 |
|--------------|------------|----------------|
| waist_cm, BMI 높음 | 🏃 운동 + 🥗 식단 | "주 3회 30분 걷기" |
| sitting_hours 높음 | 🏃 운동 | "매 1시간마다 5분 스트레칭" |
| smoking_current 높음 | 🧘 스트레스 | "금연 D-day + 호흡법 5분" |
| triglycerides, LDL 높음 | 🥗 식단 | "포화지방·당류 제한 7일" |
| HDL 낮음 | 🏃 운동 | "주 4회 유산소 운동" |
| walking_days 낮음 | 🏃 운동 | "주 5일 걷기 달성" |
| hemoglobin 낮음 | 🥗 식단 | "철분 식품 섭취 늘리기" |
| drinking_freq 해당 | 🥗 식단 | "음주 횟수 주 1회 줄이기" |
